# MySolutionDptNet6

This repository contains a basic **.NET 6 MVC** project created using Visual Studio.

## Project Structure

- **MySolutionDotNet6.sln** — Solution file
- **MyMvcApp/** — The main ASP.NET Core MVC project folder

## Getting Started

### Prerequisites

- [.NET 6 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/6.0)
- Visual Studio 2022 (recommended) or any compatible IDE

### Build and Run

1. Clone the repository:
   ```bash
   git clone https://github.com/HasanJaved-Developer/MySolutionDotNet6.git

2. Open the solution file MySolutionDotNet6.sln in Visual Studio.

3. Build the solution.

4. Run the project (usually by pressing F5 or using IIS Express).

The project will start a local web server, and you can access the default home page in your browser.
